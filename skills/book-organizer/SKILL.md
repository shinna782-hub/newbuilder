# 图书整理 (book-organizer)

> 状态：早期版本。首次使用前，请按自己的飞书多维表格字段完成配置。

## 触发指令
`图书整理`

## 工作流程

### 1. 读取图书照片信息

使用当前 Agent 可用的图片识别工具分析图书照片，正面识别书名、作者、出版社；反面识别 ISBN、标价、系列册数。

**必须读取的正反面信息：**
- 正面：书名（主标题）、书名（副标题）、作者、出版社
- 反面：系列册数（如"全套共5册"）、标价、ISBN

### 2. 系列书判断规则

当多本图书满足以下条件时，判断为同一系列：
- 书名包含相同的系列关键词（如"哈利·波特"、"三体"、"射雕英雄传"）
- 作者相同
- 出版社相同
- 封面设计风格相似

系列册数字段填写格式：如"全套共5册"或"第2册"。

### 3. 写入飞书多维表格

**目标表格：**
- app_token: 环境变量 `BOOK_ORGANIZER_BITABLE_APP_TOKEN`
- table_id: 环境变量 `BOOK_ORGANIZER_BITABLE_TABLE_ID`
- 视图: 由使用者自行配置

**字段映射：**

| 字段名 | 类型 | 来源 | 说明 |
|--------|------|------|------|
| 书名（主标题） | Text | 正面 | 书名主标题 |
| 书名（副标题） | Text | 正面 | 副标题（如有） |
| 作者 | Text | 正面 | 作者名 |
| 出版社 | Text | 正面 | 出版社名 |
| 系列册数 | Text | 反面 | 如"全套共5册"、"第2册" |
| 标价 | Text | 反面 | 如"¥128.00" |
| 书评评分 | Number | 用户补充 | 用户提供则填 |
| 书评链接 | URL | 用户补充 | 用户提供则填 |
| 记录时间 | DateTime | 自动 | 自动填充当前日期 |
| 语言分类 | SingleSelect | 推断 | 英语/双语/中文 |
| 书单分类 | SingleSelect | 推断 | 小说/纯文学/经济等 |

**语言分类选项：** 英语、双语、中文
**书单分类选项：** 小说、纯文学、经济、哲学、军事、人物、历史、社会、天文、物理、画册

### 4. 执行录入

优先使用当前 Agent 已获授权的飞书多维表格工具创建记录；也可通过飞书开放 API 写入。凭据从 `FEISHU_APP_ID`、`FEISHU_APP_SECRET` 读取，表格位置从上述环境变量读取，禁止写死在 Skill 中。`记录时间`字段使用自动填充（不填或忽略）。

## 关键提示

- 图片中无法识别的字段留空，不编造
- 一张照片 = 一本图书（除非用户说明是一套）
- 系列书需要分别录入，每本都是独立记录
- 标价格式统一保留货币符号（如"¥128.00"）
- 语言和书单分类根据书名和内容推断
